Frequently Asked Questions

How accurate are the predictions?
AeroPredict leverages an advanced LightGBM model trained on a comprehensive historical flight record dataset. It compiles airport density schedules and live weather warnings (precipitation, wind intensity, temperatures) to ensure accurate, granular risk predictions.
Where is the weather data sourced from?
Weather values are gathered directly from the Open-Meteo Weather API using coordinates for each origin airport. If the meteorological server is offline, the system falls back to seasonal averages.
What should I do if my flight is flagged high risk?
If your delay probability exceeds 50%, we recommend installing the carrier app for real-time notifications, traveling with carry-on bags to preserve re-routing flexibility, and reviewing the smart recommendations in the Travel Guide.
